# HOSxP Data Integration Pipeline
### Kamphaeng Phet Municipality Community Hospital

**Project Impact:** Optimized SQL architecture for **400,000+ patient records**, leveraging AI-driven auditing and high-performance SQL engineering to support executive decision-making.

---

## Technical Summary
* **Core Stack:** SQL (HOSxP Database), AI-Powered Data Auditing
* **Compliance:** 100% PDPA (Thailand) aligned via SQL-level Data Minimization

---

## Project Execution & Data Utilization

| Focus Area | Technical Implementation | Organizational Impact |
| :--- | :--- | :--- |
| **Executive Support** | Engineered SQL Views and Analytical Queries for BI. | Enabled **Top-Level Management** to make data-driven policy & resource decisions. |
| **AI Data Quality** | Integrated AI models to detect anomalies and invalid data patterns. | Identified and corrected complex data errors that standard SQL filters missed. |
| **Operational Auto** | Developed SQL Triggers and Stored Procedures for staff alerts. | Eliminated redundant data entry and improved staff response times. |
| **Data Security** | Enforced HN Primary Key and strict SQL schema constraints. | Secured **400K+ records** in compliance with national PDPA standards. |
